Projects & Tasks Manager by Dakshika Srivastava

This is a demo website for my submission for the TAMU AgriLife Software Application Developer interview.

This website has been created using Django on the backend. It interacts with the Project and Task databases. You can select from the following options to perform other actions.

Projects related actions:

Action Description


The "List Projects" option allows you to see a list of all the current projects in the database. This is a quick view, and only shows the project name and description. If you want a detailed view, check the "View Project" option instead.


The "View Project" option will let you view the details of whichever project you select. And this selection can be made on the next page, once you click on the button.


The "Create Project" option will let you add a new project to the database.


The "Modify Project" option will let you update the information about whichever project you select. And this selection can be made on the next page, once you click on the button.


The "Delete Project" option will let you delete whichever project you select. And this selection can be made on the next page, once you click on the button.
